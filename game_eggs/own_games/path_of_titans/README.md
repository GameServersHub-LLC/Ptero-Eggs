# Path of Titans

## Supported architecture

**ONLY** supports **amd64**.

## Server Ports

Ports are required to run the server in a table format.

| Ports             | Default |
| ----------------- | ------- |
| Game              | 25000   |
| Reserved (Game+1) | 25001   |
| RCON     (Game+2) | 25002   |
| Stats    (Game+3) | 25003   |
| Query    (Game+4) | 25004   |
